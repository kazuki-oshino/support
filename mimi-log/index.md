# みみログ / MimiLog

耳鳴り・めまいの症状を記録し、トリガーを特定するためのアプリです。

A symptom tracking app for tinnitus and dizziness to identify triggers.

---

## 主な機能 / Features

### 無料機能 / Free Features
- **症状記録** - 耳鳴り・めまいの強度、種類、持続時間を記録
- **カレンダー表示** - 月間カレンダーで症状の発生パターンを確認
- **簡易グラフ** - 直近7日間の症状推移を可視化

### プレミアム機能 / Premium Features
- **トリガー記録** - 睡眠、食事、ストレス、服薬の記録
- **気圧連動** - 気圧変化と症状の関連を確認
- **高度なグラフ** - 期間選択、統計分析、曜日/時間帯別傾向
- **相関分析** - トリガーと症状の相関を分析
- **PDFレポート** - 通院時に医師に見せるレポートを出力
- **バックアップ** - データのエクスポート/インポート
- **広告非表示**
