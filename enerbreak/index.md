# EnerBreak

エナジードリンク摂取習慣を管理し、健康的な生活を目指すアプリです。

Energy drink intake management app to help achieve healthier lifestyle goals.

---

## 主な機能 / Features

### 無料機能 / Free Features
- **衝動・摂取の記録** - ワンタップで簡単に記録
- **基本統計** - 日次摂取数、衝動回数、支出追跡
- **門限機能** - 夜間摂取を警告
- **週間チャート** - 7日間のトレンドを可視化
- **履歴表示・編集** - 過去の記録を確認・修正
- **ホーム画面ウィジェット** - 本日の摂取数をすぐに確認

### Pro機能 / Pro Features
- **カレンダービュー** - 月間パターンを可視化
- **週間/月間レポート** - 詳細な分析レポート
- **高度なチャート** - 月間・曜日・時間帯別分析
- **広告非表示**
- **Proウィジェット** - 連続回避日数・月間支出
