# App Support

アプリのサポートページとプライバシーポリシーを管理するリポジトリです。

## 構成

```
support/
├── _config.yml          # GitHub Pages設定
├── index.md             # トップページ
└── daystacks/           # DayStacksアプリ
    ├── privacy.md       # プライバシーポリシー
    └── support.md       # サポートページ
```

## GitHub Pagesの設定

1. GitHubでこのリポジトリをPublicで作成
2. Settings → Pages → Source を `main` ブランチの `/ (root)` に設定
3. 公開URLは `https://kazuki-oshino.github.io/support/` になります

## URL

- DayStacks プライバシーポリシー: `https://kazuki-oshino.github.io/support/daystacks/privacy`
- DayStacks サポート: `https://kazuki-oshino.github.io/support/daystacks/support`
