# DayStacks

A simple task manager that helps you focus on today.

---

## Features

- **Card-based tasks** - Clean, simple task cards for each day
- **Swipe navigation** - Swipe left or right to move between days
- **Optional time scheduling** - Set times if you need them
- **Color coding** - Organize tasks with colors
- **Dark mode** - Easy on the eyes
